# Changelog

All notable changes to mango are documented in this file. The format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and the project adheres
to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Project scaffolding: CMake build, MIT license, contribution docs.
- A semantic-version engine (`SemVer`) with parsing, ordering, and caret/range
  constraint matching.
- A minimal JSON reader and the `mango.json` manifest model.
- The `mango` CLI with `init`, `add`, `remove`, `tree`, `install`, `search`, and
  `publish` command dispatch.
- Documentation of the manifest format and the resolution algorithm.
